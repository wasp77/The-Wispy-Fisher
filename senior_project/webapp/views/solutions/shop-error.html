Shop Page Error

If you have tried the search functionality, then you will have noticed it affects what products show on the page. This is done through some SQL query to the database holding the products. You can test this theory by adding some SQL into your search like a single quotation mark and you will see an error message

This error affirms that it is using a SQL query and that the user input may not be validated correctly. This is a clear security misconfiguration and points to the search function most likely being vulnerable to SQL Injection.
