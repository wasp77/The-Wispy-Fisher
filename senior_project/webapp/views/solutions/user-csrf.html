Changing Username CSRF

When changing your username, you may notice a CSRF token saved as a hidden input
If you tamper with the token by say removing some values from it or even removing it altogether, you will see that it does not affect your ability to change username. This means that the server is not checking for the existence of the token when submitting a new username. To demonstrate how to exploit this vulnerability in the payloads directory there is a file called username-csrf.html if you load this HTML page by double-clicking on it you will see an "OK" message on the screen now if you navigate back to your account page you will see that your username was changed to "HACKED!".
