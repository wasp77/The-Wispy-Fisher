Product Page CSRF

When viewing a product, you will see a CSRF token added as a hidden input

if you try to remove values from it or the whole element, you will get an alert

and will not be able to make the purchase. However, you can change the individual values of the token just as long as you do not remove any. This means that the server is checking that the token is present and that it has a certain length but not the actual values. To demonstrate how to exploit this run the buy-csrf.html file found in the payloads directory. You will see an "OK" message on the screen, and if you navigate back to your account page, you will see in your purchase history that you bought the standard fly rod without knowing it.
