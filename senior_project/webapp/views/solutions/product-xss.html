Product Page XSS

The product page is vulnerable to persistent XSS. Try writing a review, and you will see that it is immediately placed on the page. If you check the element that holds the review you can see it is a li element. Therefore, the payload </li><script>alert("XSS")</script> entered as a review will trigger the XSS.

You can see how this is persistent because now every time you visit that product page you will see the XSS payload trigger.
